Thoughts on Web Accessibility:

  • Web accessibility means that everyone can access information and services on the web no matter if they have a disability or anything else that may prevent a user from viewing information on the web.
  • Web accessibility is making sure that your web page follows all the laws put in place by a country that requires a web page to be accessible for all users to avoid any legal issues or fines.
  • By making a website accessible it can reach more people, which increases user engagement on that website.
  • Accessible websites make sure that websites can remain usable and up-to-date over time.

Thoughts on Progressive Enhancement

Progressive enhancement is a web development persepctive of providing a basic functional experience that works across all devices and browsers. It is important because makes sure that users on older devices or with disabilities can still access essential content and functionality.
